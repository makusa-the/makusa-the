## 👋 Hey, I'm Innocent

I'm a **Transmission Engineer** and **Full-Stack Developer** based in 📍 Lusaka, Zambia 🇿🇲 — working across the intersection of **telecommunications networks** and **modern web platforms**.

By day I keep networks alive and optimize transmission infrastructure. By night (and often by day too) I build scalable, resilient web systems. I care deeply about the full stack — from the physical layer up to the API layer.

> *"The best engineers understand both the wire and the code that runs on it."*
